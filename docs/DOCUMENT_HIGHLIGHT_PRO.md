# Visor documental profesional — Smart Filter 1.0.20-dev

## Dependencia

Requiere SharedCode Cores 1.0.0, instalado desde la Release pública validada.

## Flujo

1. Smart Filter vuelve a leer el archivo seleccionado bajo demanda.
2. Convierte PDF, DOCX, XLSX, CSV o texto a un modelo neutral.
3. Entrega el contrato estándar a RenderCore.
4. RenderCore utiliza el perfil `document_highlight_pro`.
5. El navegador abre una copia HTML temporal; el original permanece intacto.

## Experiencia

- Cabecera y métricas con el lenguaje visual del ecosistema.
- Sidebar sticky con filtro, términos, ubicaciones, contenido y acciones.
- Navegación Anterior/Siguiente con progreso y ubicación actual.
- Atajos de teclado izquierda/derecha.
- Secciones plegables para páginas, hojas y bloques.
- Resaltado tolerante a mayúsculas y tildes.

## Compatibilidad

La búsqueda agrupada por categoría, los contadores, las exportaciones y la acción Abrir original no cambian.
